# 📦 Сервис отслеживания посылок

Консольное приложение на Golang для управления посылками с хранением данных в SQLite.

## 🔧 Возможности

- Регистрация новой посылки
- Получение списка посылок клиента
- Изменение статуса посылки
- Изменение адреса доставки
- Удаление посылки

## 📦 Статусы посылок

| Статус | Значение |
|--------|----------|
| `registered` | Зарегистрирована |
| `sent` | Отправлена |
| `delivered` | Доставлена |

## 🗄️ Структура БД

Таблица `parcel`:
- `number` — номер посылки (автоинкремент)
- `client` — ID клиента
- `status` — статус посылки
- `address` — адрес доставки
- `created_at` — дата создания

## 🛠️ Стек

- Golang
- SQLite
- Стандартная библиотека (`database/sql`)
